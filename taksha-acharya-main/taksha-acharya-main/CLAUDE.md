# taksha-acharya-app — Production App

## What This Is
Taksha Workshop Carpentry Skill training agent — internal trilingual playbook (Bengali/Hindi/English), AI chat + quiz + voice.

## Tech Stack
- **Framework:** Next.js 16 (App Router, TypeScript)
- **Database:** Supabase (shared Taksha instance, tables prefixed `taksha_`)
- **Styling:** Tailwind CSS v4
- **State:** Zustand (persisted to localStorage)
- **AI:** Claude Sonnet (chat), Claude Haiku (quiz), Google TTS (voice)
- **Deploy:** Vercel

## Key Architecture Decisions
- Global module + language selectors in header — all tabs read from zustand store
- Content stored per-section per-language in Supabase (not monolithic)
- Progress: localStorage first (instant), Supabase sync (async, Phase 2)
- API routes at `src/app/api/{chat,quiz,tts}/route.ts`
- System prompt in `src/lib/system-prompt.ts` (shared by chat route)
- Event tracking via `src/lib/learner-sync.ts` → `taksha_events` (event_data JSONB holds module_id + arbitrary payload)

## Supabase
- Migration: `supabase/migrations/001_taksha_initial_schema.sql`
- Seed: `supabase/seed-taksha-content.sql` (generated by `supabase/generate-expanded-sql.js`)
- 12 tables, all prefixed `taksha_`
